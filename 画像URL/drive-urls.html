drive → image urls

Googleドライブのフォルダ内の画像を、ファイル名つきで一覧＆URL化します。
準備（最初の1回だけ）
  1. 対象フォルダを右クリック →「共有」→「リンクを知っている全員」を閲覧者にする。
  2. console.cloud.google.com でプロジェクトを作成 →「APIとサービス」で Google Drive API を有効化。
  3. 「認証情報」→「APIキーを作成」。作ったキーをこの画面に貼る。
  4. （任意）キーは「APIの制限＝Drive API」「アプリケーションの制限＝HTTPリファラー」で自分の公開ドメインに絞ると安全。
URLは https://lh3.googleusercontent.com/d/ファイルID=w幅 形式で出します。<img> にそのまま使えます。どれがどれか分かるよう、ドライブ側のファイル名を work1.jpg のように付けておくと、そのままスプレッドシートに流し込めます。
